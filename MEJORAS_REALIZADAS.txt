Nombre:        install_smart.ps1
Propósito:     Instalador inteligente
Características:
  ✅ DETECTA si RustDesk ya está instalado
  ✅ Si está instalado → solo configura (2 minutos)
  ✅ Si no está → descarga, instala y configura (5-7 minutos)
  ✅ Verifica configuración correctamente
  ✅ Espera a que RustDesk cargue
  ✅ Muestra el contenido del archivo TOML creado
  ✅ Valida cada paso

Mejoras concretas:
  • Verifica por archivo ejecutable
  • Verifica por registro de Windows
  • Espera correctamente a que se cree la carpeta
  • Valida que el archivo se escribió correctamente
  • Muestra feedback detallado de cada paso
  • Mejores tiempos de espera entre pasos


Nombre:        configure_smart.ps1
Propósito:     Configurar servidor en RustDesk existente
Características:
  ✅ Verifica que RustDesk está instalado
  ✅ Detiene RustDesk antes de configurar
  ✅ Crea la carpeta de configuración si no existe
  ✅ Escribe el archivo TOML correctamente
  ✅ Verifica y muestra el contenido
  ✅ Reinicia RustDesk
  ✅ Instrucciones de verificación al final

Mejor para:
  • Cambiar servidores sin reinstalar
  • Reconfigurar si algo no funcionó bien

─────────────────────────────────────

Comando:
  .\install_smart.ps1

Qué hace:
  1. Verifica si RustDesk está instalado
  2. Si NO está → descarga, instala y configura (5-7 min)
  3. Si SÍ está → solo configura (2 min)
  4. Valida la configuración
  5. Inicia RustDesk
  6. Muestra instrucciones finales


CAMBIAR CONFIGURACION DESPUES:
─────────────────────────────

Comando:
  .\configure_smart.ps1

Qué hace:
  1. Verifica que RustDesk está instalado
  2. Detiene RustDesk
  3. Modifica la configuración
  4. Valida y muestra el archivo
  5. Reinicia RustDesk
  6. Instrucciones para verificar

   ───────────────────────
   Método 1: Verifica si existe %APPDATA%\RustDesk\rustdesk.exe
   Método 2: Busca en HKLM:\Software\Microsoft\Windows\...Uninstall
   Resultado: Si al menos uno existe → RustDesk está instalado

2. VALIDACIÓN DE CONFIGURACIÓN
   ──────────────────────────
   • Escribe el archivo TOML
   • Verifica que el archivo existe
   • Lee el contenido y lo muestra
   • Verifica que está en UTF-8

3. TIEMPOS DE ESPERA MEJORADOS
   ──────────────────────────
   • Espera a que se cree la carpeta de configuración
   • Espera a que RustDesk termine de instalar
   • Espera a que RustDesk cargue la configuración
   • Esperas entre pasos (para evitar conflictos)

4. MANEJO DE ERRORES
   ────────────────
   • Try-catch en secciones críticas
   • Valida rutas antes de usarlas
   • Verifica permisos
   • Muestra errores específicos al usuario
